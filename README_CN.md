## ![160x160](./Doc/160x160.png)

[OneJoystick]( https://assetstore.unity.com/packages/slug/189556 
) 是一个简单易用可以高度定制化的游戏虚拟摇杆。

![420x280](./Doc/420x280.png)



## 特点

- 内置遥感在2D和3D场景下的使用案例。
- 支持Drag，Update，LateUpdate下的拖动回调的设置。
- 代码简洁：可以高度定制化。
- 支持遥感箭头的跟随显示。
- 一个脚本文件包含了所有功能。
